Get Started in 30 Seconds

Install SudoDog and Docker, then run any agent securely.

# Install SudoDog
curl -sL https://sudodog.com/install.sh | bash

# Install Docker (optional, for stronger isolation)
curl -fsSL https://get.docker.com | sudo sh

# Initialize
sudodog init

# Run with Docker isolation
sudodog run --docker python your-agent.py

# Start background monitoring
sudodog daemon start

Works on Ubuntu, Debian, Arch, Fedora, and most Linux distributions.
